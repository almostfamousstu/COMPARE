Welcome to SKU Specification Comparator

The SKU Specification Comparator is a modern web application that helps you validate and compare product specifications against AI-generated data from Google Gemini. Built with Next.js 14 and powered by Google's Gemini AI, it provides instant visual feedback on specification discrepancies.

What is SKU Comparator?

SKU Comparator is a sleek, dark-mode-first application designed to streamline the process of validating product specifications. Simply provide a SKU identifier and your product attributes, and the application will query Google Gemini to generate corresponding specifications, highlighting any differences between your data and the AI's interpretation.

Key Capabilities

  • AI-Powered Comparison: Leverage Google Gemini 2.5 Flash to generate specification data
  • Flexible Input: Accept data in JSON format or simple colon-separated lines
  • Visual Diff Highlighting: Instantly identify mismatches with color-coded results
  • Structured Output: Ensure consistent responses using JSON schema validation
  • Dark Mode Interface: Modern, eye-friendly design optimized for extended use
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices

How It Works

  1. Input Your Data

    Enter a SKU identifier and paste your product attributes in JSON or colon-separated format.

  2. AI Analysis

    The application sends your attribute keys to Google Gemini, which generates corresponding values based on its knowledge of the SKU.

  3. Visual Comparison

    Results are displayed in an interactive table with clear highlighting of any differences between your values and the AI's values.

  4. Filter & Review

    Toggle to show only differences or review all attributes side-by-side to make informed decisions.
